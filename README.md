# vue-cli 3 + ts addRouter 动态添加路由 实现用户访问权限

## 配置

```
vue-cli3 + mockJs + ts + vue-router + vuex + element-ui + axios + addRouter
```

mock    接口模拟目录
router  路由分配 以及 router.beforeEach
